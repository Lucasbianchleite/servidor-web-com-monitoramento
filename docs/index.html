Servidor Web com Monitoramento Automático

Projeto focado em infraestrutura Linux com Nginx, automação de verificações periódicas e envio de alertas em tempo real para canais como Discord, Telegram e Slack. A proposta é garantir visibilidade, resposta rápida a falhas e uma base sólida para estudos de DevOps, observabilidade e operações.

📘 Visão Geral do Projeto

Fluxo do Projeto

Um fluxo simples, direto e fácil de explicar em apresentação ou README.

1

Servidor Nginx em execução

O serviço web publica a aplicação HTML e responde às requisições HTTP do ambiente.

2

Script realiza checagens periódicas

Uma rotina automatizada testa o acesso ao site em intervalos definidos.

3

Falhas são registradas

Se houver erro de conexão, timeout ou resposta inválida, o evento é salvo em log.

4

Notificação é enviada

O time recebe um alerta imediato por Discord, Telegram ou Slack para agir rapidamente.
